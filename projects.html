JustWork

This is most recent project I worked on for a hackathon. I worked with a group of my friends in total there was four of us. Using the language flutter we developed a mobile app that works as a punch card you hit a button when you start your shift and at the end of your shift you type in the work you completed and punch out. Using a specialized code you can then link your account with your employer so they can keep track of your hours and read your descriptions. We didn't finish the app but we created the punch in punch out page and a page that connects to a google firebase server to display the submissions.

Side Scroller and Adventure Game

A simple Java project that uses the basic graphics utilities provided by Java libraries to draw images to a window. To make this I followed an online tutorial but I couldn't find the creator to credit them as this was made years ago. Since back when this picture was taken I have adjusted that engine numerious times. The changes I made added things such as sound and easier ways to create levels. I combined Side Scroller with Adventure Game as they are made in the same engine. The only difference is that Side Scroller is of an action platformer design while Adventure Game is top down. A Java project.

Maze Game

A simple console game that I hand coded my first semester in college. It consists of characters used to portray a hall that splits off in four directions. The goal of the game is to make it to the end of the maze by navigating it by typing keywords in the console. It was the first program I had written without any outside help. A Java project.
